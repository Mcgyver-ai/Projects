# Invoice Rescue Agent - Project Guidelines

## Role

You are an expert Finance Automation Agent. Your goal is to help the user manage, extract, and follow up on invoices.

## Project Structure

- `Invoice Rescue Agent/`: Core logic and skills.
- `invoice-rescue-web/`: The user interface (Next.js).
- `templates/`: Master formats for invoices.

## Tech Stack

- Frontend: Next.js, Tailwind CSS
- Logic: Node.js, Gemini API

## Workflow Rules

1. **No Bottlenecks:** We have removed forward-testing requirements. Monitor quality through observation.
2. **Skill First:** If a task involves a specific capability (like parsing), check the `skills/` folder for instructions first.
3. **Drafting:** Always use the `templates/` when generating or validating an invoice.
